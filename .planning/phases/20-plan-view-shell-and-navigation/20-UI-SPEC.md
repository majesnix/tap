## Interaction Contracts

### Plans Nav Button (in Sidebar)

- **Position**: Top of sidebar, before FileSection, after the "Tap" title + tagline block.
- **Icon**: `ListChecks` (lucide) — represents a multi-step plan.
- **Label**: "Plans" — text label beside the icon.
- **Default state**: `variant="ghost"` Button with icon + label.
- **Active state (viewMode === "plans")**: `bg-accent text-accent-foreground` — same token as selected plan row.
- **Inactive state**: `hover:bg-muted` on hover.

### PlanView — Two-Pane Layout

- **Outer container**: `flex h-screen w-screen overflow-hidden bg-background text-foreground` — mirrors AppLayout.
- **Left pane (PlanListPanel)**: `w-72 shrink-0 border-r border-border flex flex-col h-full`.
- **Right pane (PlanDetailPanel)**: `flex-1 flex flex-col h-full` — placeholder only in Phase 20.

### Plan List Rows

- **Unselected + unhovered**: `bg-transparent`.
- **Hovered (unselected)**: `hover:bg-muted`.
- **Selected**: `bg-accent text-accent-foreground`.
- **Selected + hovered**: `bg-accent text-accent-foreground` (selection takes precedence).
- **Kebab button**: Always visible (not hover-gated) — `variant="ghost" size="icon-sm"` — `MoreVertical` (lucide) icon.
- **Click anywhere on row (except kebab)**: Sets `selectedPlanId`.

### Inline Create Row (D-07)

- Triggered by "New Plan" button in PlanListPanel header.
- Renders at **bottom of the plan list** (after all existing rows) as a controlled `<Input>` with `autoFocus`.
- Pre-filled value: `"Untitled Plan"` with all text selected on mount (`ref.select()` in useEffect).
- **Enter key**: Commit — calls `usePlanStore.createPlan(name.trim())`. Row replaced by real plan row.
- **Escape key**: Cancel — removes the pending row without creating a plan.
- **onBlur**: Same as Escape (discard) — prevents orphaned rows on focus loss.
- **Empty-name guard (D-09)**: If trimmed value is empty, treat as Escape.

### Inline Rename Row (D-08)

- Triggered by "Rename" in the kebab DropdownMenu.
- Replaces the plan name `<span>` in the row with an `<Input autoFocus>` pre-filled with the current plan name.
- **Enter key**: Commit — calls `usePlanStore.renamePlan(id, name.trim())`.
- **Escape key**: Discard — restores the static plan name.
- **onBlur**: Same as Escape.
- **Empty-name guard**: Trim empty = Escape (discard, no rename).

### Kebab DropdownMenu Items

Listed in this exact order:
1. **Rename** — `Pencil` icon (lucide), normal variant.
2. **Duplicate** — `Copy` icon (lucide), normal variant.
3. **Delete** — `Trash2` icon (lucide), `text-destructive` color.

### Delete AlertDialog (D-05)

- Title: `Delete "{plan name}"?`
- Description: `"This action cannot be undone."`
- Cancel button: `"Keep plan"` — `variant="outline"`.
- Confirm button: `"Delete plan"` — `variant="destructive"`.

### Right Pane Empty States (D-03)

- **No plan selected**: Centered vertically + horizontally, `text-muted-foreground`.
  - Heading (text-sm font-medium): `"Select a plan to view its steps"`
  - Body (text-xs): `"Click a plan in the list to see its steps here."`
- **Plan selected, no steps**: Same centered layout.
  - Heading (text-sm font-medium): `"No steps yet"`
  - Body (text-xs): `"Steps will appear here once you add them in the next phase."`
