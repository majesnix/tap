    collectFieldNames(obj: Record&lt;string, unknown&gt;): string[]
    - for each [key, value] in Object.entries(obj): if key === "_selected" continue; push key
    - if value !== null AND !Array.isArray(value) AND typeof value === "object": recurse via collectFieldNames
    - else if Array.isArray(value): for each item in value, if item !== null AND typeof item === "object" AND !Array.isArray(item): recurse
    - primitives: push key only, no recursion
    - returns flat string[] of all collected keys

    filterHistoryEntries signature extension:
    - Add 4th parameter: searchQuery = ""
    - Existing 2-filter chain is unchanged
    - Chain a third .filter() stage after the existing two:
      - if !searchQuery return true (empty = pass-all)
      - const q = searchQuery.toLowerCase()
      - if e.messageTypeName.toLowerCase().includes(q) return true
      - if e.exchange.toLowerCase().includes(q) return true
      - if e.routingKey.toLowerCase().includes(q) return true
      - const fieldNames = collectFieldNames(e.fieldValues); return fieldNames.some(name => name.toLowerCase().includes(q))
  </behavior>
