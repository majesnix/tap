## Notes on Requirement Outcomes

- **CICD-01**: Achieved via tauri-action with `releaseDraft: true`; signing env vars conditionally applied when on a tag push.
- **SIGN-01/02**: Required restoring `cs.allow-unsigned-executable-memory` — originally removed by a security review pass; WKWebView JIT requires it under Hardened Runtime.
- **UPD-02/03**: Root cause of initial failure was private repository. `tauri-plugin-updater` makes unauthenticated HTTP requests; the 404 was silently swallowed. Fixed by making the repo public.
- **Bonus (not originally in scope)**: `Check for Updates...` in macOS native menu bar + sidebar button for Windows/Linux — delivered this session.
